OS笔记-CH-8-死锁

死锁产生的条件

死锁：当一组进程中的每个进程都在等待某个事件（资源），而仅有这组进程中被阻塞的其他进程才可触发该事件时，则认为该组进程发生了死锁。

  • 死锁是永久性的，死锁问题没有有效而通用的解决方案

资源

  • 可重用资源
    • 主存、处理器、IO通道
  • 消费型资源
    • IO buffers

进程使用资源的正确顺序：申请、使用、释放

必要条件

  • 互斥
  • 占有等待
    • 当一个进程等待其他进程时，继续占有已分配资源
  • 非抢占
  • 循环等待

处理死锁的方法

  • 确保系统永远不会进入死锁状态

    • 死锁预防

      ​ 使死锁的四个必要条件之一无效

      • 互斥 -> 可共享资源不需要互斥
      • 占有等待 -> 不许占着等。
        • 开始执行之前请求分配所有资源，仅当进程完全没有资源的时候再分配
        • 可能会饥饿，利用率低
      • 非抢占 -> 有得不到的资源就释放其他的，排队等着去
      • 循环等待 -> 按类型强制排序所有资源

    • 死锁避免

  • 允许进入死锁状态，然后检测恢复

  • 默认不发生

TOEFL_notes_学术讨论（模板结构）

要求

  • 10min

  • at least 100 words

  • recommend: 130-150 words

  • topic development 观点明确 一条路走到黑

常见考题：school education / everyday matters / jobs / government and social matters

组织结构

  • main topic（明确）
    • 和人物观点重复：as xxx has already mentioned, …（不一定要加）
    • 在我看来：i think / in my opinion / i believe / in my point of view / personally / from my perspective / in my eyes
  • supporting idea1 + explanation (general expressions) + example (s)（有细节！！！）
    • 第一：to begin with / to start with / initially / primarily…
    • 举例子：for example / for instance
  • supporting idea2 + explanation + example (s)
    • 第二：furthermore / what’s more / besides / additionally / in addition…
  • conclusion
    • 总结：In summary / all in all / to sum up / in conclusion… 注：不能和开头句的表达一模一样，paraphrase

语言

  • 关键的
    • important: essential; critical; crucial; vital; significant; sth matters more…
  • 能力
    • ability / skill / ability / capacity e.g.社交；沟通；合作；public speaking
  • 好处
    • better / effective / advantageous / beneficial phr. bring / create benefits / advantages
  • 坏处
    • harmful / meaningless
  • 非常
    • very / really / extremely / incredibly…
  • 不放句首
    • for / and / nor / but / or / yet / so
  • 因此
    • thus / therefore / hence / as a result / consequently

例子

e.g.
project-based learning
-more impressive. -Doing project helps learning the knowledge well- e.g. artificial intelligence: don’t know how to debug; after doing the project-totally skilled
-ability up -doing project help know more methods to solve problem and also improve communicate skill - e.g. facing problem: ask friends and search

e.g.
From my perspective, going on field trips is an essential part of children’s education.

To start with, as Paul has already mentioned, field trips are able to make a difference on kids’ future. During a field trip, children will have a possibility of exploring and learning new knowledge, which might help cultivate their interest of a particular thing. When I was young, I entered a science museum when went on a field trip and was attracted by the astonishing robots there, which convinced me of taking computer science as my major.

Furthermore, kids and teachers are both able to take a break by going on field trips. In daily studies, the courses are so tough that students and teachers are always exhausted, taking field trips gives them a day to relief, which helps them concentrate on their work and studies in the future. When I was a kid, I was always looking forward to field trips because the studies made me tired and I wanted to relax.

In conclusion, field trips are necessary in kids’ education.

新要求

  • 不要写复杂地要死的句子
  • 不要举背下来的例子

备考方法

  • 5min outline （2层分论点+e.g.）
  • 10min finish the paragraph（超时标记最终完成时间）
  • 5min review the article -打开修订标记
  • 标点符号修改；单复数；三单；时态使用；拼写etc.
