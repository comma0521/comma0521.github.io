特征工程

0. 关于特征工程的简单介绍

Feature engineering is a machine learning technique that leverages data to create new variables that aren’t in the training set. It can produce new features for both supervised and unsupervised learning, with the goal of simplifying and speeding up data transformations while also enhancing model accuracy. Feature engineering is required when working with machine learning models. Regardless of the data or architecture, a terrible feature will have a direct impact on your model.

特征工程是一种机器学习技术，它利用数据创建不在训练集中的新变量。它可以为监督和非监督式学习生成新的特性，目的是简化和加快数据转换，同时提高模型的准确性。在处理机器学习模型时需要特征工程。无论是数据还是架构，一个糟糕的特性都会对您的模型产生直接的影响。

特征：原始数据某个方面的数值表示，特征是数据和模型之间的纽带。

定义：从原始数据中提取特征并将其转换为适合机器学习的格式，从而改善机器学习性能（提高预测准确度），同时减少运行时间。

目的：机器学习的一种优化，提高机器学习性能

流程：

Feature engineering consists of various processes:

  • Feature creation（创建）: Creating features involves creating new variables which will be most helpful for our model. This can be adding or removing some features.
  • Transformations（转换）: Feature transformation is simply a function that transforms features from one representation to another. The goal here is to plot and visualize data. If something isn’t adding up with the new features, we can reduce the number of features used, speed up training or increase the accuracy of a certain model.（提高准确性，通过变换or映射实现，可视化）
  • Feature extraction（提取）: Feature extraction is the process of extracting features from a data set to identify useful information. Without distorting the original relationships or significant information, this compresses the amount of data into manageable quantities for algorithms to process.（中文意思就是把数据集整理成算法可以处理的样子，压缩）
  • Exploratory data analysis（探索性分析） : Exploratory data analysis (EDA) is a powerful and simple tool that can be used to improve your understanding of your data, by exploring its properties. The technique is often applied when the goal is to create new hypotheses or find patterns in the data. It’s often used on large amounts of qualitative or quantitative data that haven’t been analyzed before.
  • Benchmark : A benchmark model is the most user-friendly, dependable, transparent and interpretable model against which you can measure your own. It’s a good idea to run test data sets to see if your new machine learning model outperforms a recognized benchmark. These benchmarks are often used as measures for comparing the performance between different machine learning models like neural networks and support vector machines, linear and non-linear classifiers or different approaches like bagging and boosting. 基准模型是一种用户友好、可靠、透明且易于解释的模型，用作衡量和比较新机器学习模型性能的标准。通过对测试数据集的分析，基准模型可以帮助你判断新开发的机器学习模型是否超越了公认的性能标准。这些基准通常用于比较不同机器学习模型（如神经网络与支持向量机）、不同类型的分类器（线性与非线性）或不同的方法（如Bagging与Boosting）之间的性能。简言之，基准模型为机器学习模型的性能评估提供了一个参考点，有助于研究人员或开发人员评估和优化他们的模型。

1. 对数据集的各种最初始的处理

特征增强（Feature Enhancement）

它指的是通过各种技术改善或增加数据集中特征的质量和有效性，以使机器学习模型能够从这些特征中学习到更多有用信息。特征增强的目的是提高数据的可用性，强化模型的预测能力。

preparing your data for machine learning.

缺失值处理（Missing values）

  1. Numerical imputation.（数字）

对缺失值处理主要有两种方式：

  • 删除缺失值，这种一般适用于数据集中缺失值较少的情况，如果缺失值比例超过10%，那么删除数据会对数据分析结果有很大影响，不合理，需要填充值；
  • 填充值，填充的值可以是均值、中位值、前值、后值，对于分类变量而言，可以是众数，此外还可以进行插值，如二次曲线插值，某些情况下也可以填充0，可以根据实际情况选择合适的填充值。
  1. Categorical imputation.（分类）

When dealing with categorical columns, replacing missing values with the highest value in the column is a smart solution. However, if you believe the values in the column are evenly distributed and there is no dominating value, imputing a category like “Other” would be a better choice, as your imputation is more likely to converge to a random selection in this scenario.

在处理分类列时，用列中的最高值替换缺失值是一个明智的解决方案。但是，如果您认为列中的值是均匀分布的，且不存在主导值，那么使用 “其他 “这样的类别进行估算会是更好的选择，因为在这种情况下，您的估算更有可能趋近于随机选择。

异常值处理

removing outliers from a data set

  1. Removal: Outlier-containing entries are deleted from the distribution. However, if there are outliers across numerous variables, this strategy may result in a big chunk of the datasheet being missed.（直接删，但是可能删太多了）
  2. Replacing values: Alternatively, the outliers could be handled as missing values and replaced with suitable imputation.（当成缺失值作替换）
  3. Capping: Using an arbitrary value or a value from a variable distribution to replace the maximum and minimum values.（用某个值换）
  4. Discretization : Discretization is the process of converting continuous variables, models, and functions into discrete ones. This is accomplished by constructing a series of continuous intervals (or bins) that span the range of our desired variable/model/function.↓

分箱策略：将连续数据分箱（或分段）可以有效地“平滑”数据，减少异常值对数据分析的影响。通过将连续的数据值划分到固定数量的区间内，单个的异常值对整体数据的影响被限制在其所在的箱内。这种方法可以使数据分析对异常值不那么敏感。

异常值的隔离：在离散化过程中，可以通过设定特定的区间来隔离异常值，比如为极端的数值创建专门的箱。这意味着异常值被单独分组，而不是与其他“正常”值混合，有助于后续分析中识别和处理这些异常值。

减少异常值带来的偏差：连续数据中的极端值可能会对数据分析和模型预测产生不成比例的影响。通过离散化，数据被简化为几个类别，这可以减轻单个异常值可能带来的偏差。

数据的鲁棒性提高：将连续变量离散化后，数据分析和模型训练对异常值的敏感度降低，增加了模型对噪声和异常值的鲁棒性。

便于后续处理：对于一些机器学习算法，尤其是那些基于树的模型（如决策树、随机森林、梯度提升树等），离散化的变量可能更容易处理，同时使得模型更容易解释。此外，异常值的处理也变得更加直接，因为它们已经被划分到特定的箱中。

类别变量编码

特征的数据类型可分为两大类，即连续型和离散型。其中离散型特征既有数值型的，也有类别型的。例如，性别（男、女）、成绩等级（A、B、C）等。连续型特征的原始形态就可以作为模型的输入，无论是LR、神经网络，还是SVM、GBDT、xgboost等。但是除了决策树等少数模型等直接处理字符串形式的类别型特征外，逻辑回归、支持向量机等模型的输入必须是数值型特征才能进行训练。因此，就需要将离散型中的类别型特征转换成数值型的。

有顺序：序号编码（Ordinal Encoding）

序号编码通常用于处理类别间具有内在大小顺序关系的数据，对于一个具有m个类别的特征，我们将其对应地映射到[0, m-1]的整数。例如对于”学历”这样的类别，”学士”、”硕士”、”博士” 可以很自然地编码成 [0,2]，因为它们内在就含有这样的逻辑顺序。但如果对于“颜色”这样的类别，“蓝色”、“绿色”、“红色”分别编码成[0,2]是不合理的，因为我们并没有理由认为“蓝色”和“绿色”的差距比“蓝色”和“红色”的差距对于特征的影响是不同的。

没顺序： 独热编码（One-hot Encoding）

如果类别特征本身有顺序（例：优秀、良好、合格、不合格），那么可以保留单列自然数编码，但是如果类别特征没有明显的顺序（例：红、黄、蓝），则可以使用独热编码（One-Hot Encoding），即每个类作为1个特征。输出的矩阵是稀疏的，含有大量的0。通过独热编码，每个类别特征被转换成一个二进制（0或1）向量，其中只有一个位置是1（表示该类别），其余都是0。

特征变换：把数据变得更整齐

对数变换（最常用）

Log transform is the most used technique among data scientists. It’s mostly used to turn a skewed distribution into a normal or less-skewed distribution. We take the log of the values in a column and utilize those values as the column in this transform. It’s used to handle confusing data, and the data becomes more approximative to normal applications.

常见的数据预处理技术，通常用于减小数据的偏斜（skewness）和使数据更接近正态分布。

原理：

  1. 缩小极端值的影响

对数变换可以显著减少数据中极端值（大数值）的影响，而对较小的数值影响较小。这是因为对数变换是非线性的，它对较大的数值进行了“压缩”，而对接近0的数值变化不大。这种压缩效果减少了数据的偏斜程度，使得数据的分布更加接近于正态分布。

  1. 转换乘法关系为加法关系

对数变换将原始数据中的乘法关系转换为加法关系。这对于处理数据中的乘法效应（如指数增长）非常有用，因为正态分布假设数据的变异是加性的。通过对数变换，可以使得变量之间的复杂乘法关系线性化，更适合采用假设加性效应的统计模型进行分析。

  1. 改善线性关系

在回归分析等统计建模中，对数变换常用于改善自变量和因变量之间的线性关系。如果数据呈现指数增长或者具有非线性关系，对数变换可以帮助线性化这种关系，从而满足线性模型的假设。

特征转换（降维）：处理成易于学习的形式

(西瓜书第十章)

  • 降维旨在减少数据集中的特征数量，同时尽可能保留原始数据的信息。本来其实特征选择完成就可以开始训练了，但是特征矩阵太大会导致开销太大，所以先降维。
  • 常用的降维技术包括主成分分析（PCA）、线性判别分析（LDA）和t-分布随机邻域嵌入（t-SNE）等。

k近邻学习

  • 常用的监督学习方法
  • 机制：给定测试样本，基于某种距离度量找出训练集中与其最靠近的k个及训练样本，基于k个邻居的信息进行预测
    • 分类任务——投票法（k个里出现次数最多的）
    • 回归任务——平均法（取平均值做预测结果）
    • 都可以加权，基于距离
  • 没有显式的训练过程——懒惰学习的代表（训练阶段只是把样本保存，训练时间开销0，收到测试样本在处理），直接处理的交急切学习
  • 最近邻分类器，基于贝叶斯最优分类器：通过某对象的先验概率，利用贝叶斯公式计算出其后验概率，即该对象属于某一类的概率，选择具有最大后验概率的类作为该对象所属的类。

低维嵌入

  • 引入原因：高维情形出现样本稀疏、距离计算困难——维数灾难，需要缓解

  • 含义：通过数学变换将原始高维属性空间转变为一个低维“子空间”

  • 作用：提高样本密度和距离计算难度，使更容易学习

  • 方法：

    • ==多维缩放（MDS）==

    • 步骤：

    • 1. 计算距离矩阵

      • 首先，计算高维空间中所有数据点对之间的距离，生成一个距离矩阵。这个距离可以是欧几里得距离（勾股定理）、曼哈顿距离（轴距总和）或其他任何适合数据的距离度量。

      2. 配置初始化

      • 在低维空间中随机初始化数据点的位置。这些初始位置将在接下来的优化过程中被调整。

      3. 距离优化

      • MDS通过优化过程调整低维空间中点的位置，以使低维空间中的距离尽可能接近高维空间中的距离。优化过程通常使用某种形式的梯度下降方法。

      4. 度量标准

      • MDS可以是度量的（metric）或非度量的（non-metric）。度量MDS尝试保持距离的绝对值，而非度量MDS仅尝试保持距离的顺序（即，如果两个点在高维空间中是最近的，那么它们在低维空间中也应该是最近的

  • 评估方式：比较降维前后学习器的性能，可视化技术直观

主成分分析（PCA）

  • 最常用的降维方法之一

  • 核心思想：用一个超平面对所有样本进行恰当的表达

    • 最近重构性：样本点到这个超平面距离足够近
    • 最大可分性：样本点在这个超平面上的投影能尽可能分开

  • 方法：寻找数据的主要变动方向。它找到了一个新的坐标系统，这个坐标系统的基向量（称为主成分）按照数据方差的大小排序，方差越大的方向（也就是越能分得开的方向）能够捕获更多的数据信息。

标准化数据：首先，通常需要对原始数据进行标准化处理，使得每个特征的平均值为0，标准差为1。这一步是为了防止原始数据中不同的量纲对结果产生影响。

构建协方差矩阵：接着，计算数据的协方差矩阵。协方差矩阵描述了数据中各个特征之间的相互关系，以及它们各自的变动情况。

计算特征值和特征向量：对协方差矩阵进行特征分解，得到一系列特征值和对应的特征向量。特征向量指示了数据的主要变动方向，而特征值的大小表示了在对应方向上的变动程度（即方差）。

选择主成分：按照特征值的大小，选择最大的几个特征值对应的特征向量作为主成分。这些主成分构成了新的坐标系统，按照数据变异程度（方差）的重要性排序。

构造新的数据表示：最后，将原始数据投影到这些主成分上，得到降维后的数据。这些主成分构成的超平面是原始数据在低维空间中的最佳线性表示。

定类数据（Nominal Data）

  • 属性：数据是离散的且无序的。这意味着数据用于标记或分类，但分类之间没有顺序。
  • 例子：种类（如血型、性别、国籍）。
  • 描述性统计：频率（各类别出现的次数）、占比（各类别在总数中的比例）、众数（出现次数最多的类别）。
  • 图表：条形图、饼图。

定序数据（Ordinal Data）

  • 属性：数据是有序类别的，即数据的分类不仅将对象分开，分类之间还有一个明确的排序。
  • 例子：考试等级（如A、B、C等级）。
  • 描述性统计：频率、众数、中位数（将数据分为上下两半的点）、百分数。
  • 图表：条形图、饼图、茎叶图。

定距数据（Interval Data）

  • 属性：数据是数字的，且数字之间的差别是有意义的。但是没有真正的“零点”，这意味着不能对数据进行乘除运算。
  • 例子：摄氏度、年份。
  • 描述性统计：频率、众数、中位数、均值（算术平均值）、标准差（数据分布的程度）。
  • 图表：条形图、饼图、茎叶图、箱线图、直方图。

定比数据（Ratio Data）

  • 属性：数据是连续的，可以进行加减乘除运算。这是因为定比数据有一个真正的零点。
  • 例子：金钱、重量。
  • 描述性统计：均值、标准差。
  • 图表：直方图、箱线图。

某些统计测试和图表只适用于特定类型的数据。了解这些等级可以帮助研究者或数据分析师选择合适的方法来分析他们的数据。

数据分布

直方图

核密度图

核密度图本质上是根据有限的数据样本，运用核密度函数，对整体数据的密度进行估计；即已知有限的数据样本和一个核函数，输出整体数据的概率密度，并通过图形展示出结果。

箱型图

它主要用于反映原始数据分布的特征，还可以进行多组数据分布特征的比 较。箱线图的绘制方法是：先找出一组数据的上边缘、下边缘、中位数和两个四分位数；然后， 连接两个四分位数画出箱体；再将上边缘和下边缘与箱体相连接，中位数在箱体中间。

箱形图判断异常值的标准以四分位数和四分位距为基础，四分位数具有一定的耐抗性，多达25%的数据可以变得任意远而不会很大地扰动四分位数，所以异常值不能对这个标准施加影响，箱形图识别异常值的结果比较客观。由此可见，箱形图在识别异常值方面有一定的优越性。

饼图

散点图

散点图是指在回归分析中，数据点在直角坐标系平面上的分布图，散点图表示因变量随自变量而变化的大致趋势，据此可以选择合适的函数对数据点进行拟合。

用两组数据构成多个坐标点，考察坐标点的分布，判断两变量之间是否存在某种关联或总结坐标点的分布模式。散点图将序列显示为一组点。值由点在图表中的位置表示。类别由图表中的不同标记表示。散点图通常用于比较跨类别的聚合数据。（描述变量之间的关系）
