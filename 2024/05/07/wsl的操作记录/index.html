使用wsl访问window11主文件系统

WSL 自动将 Windows 的驱动器挂载到 Linux 文件系统中，通常在 /mnt/ 下。例如，如果代码在 Windows 的 C: 驱动器上，你可以在 WSL 的终端中通过 /mnt/c/ 路径找到它。例如，如果你的代码位于 C:\code\go\test.go，你可以在 WSL 中通过 /mnt/c/code/go/test.go 访问此文件。
